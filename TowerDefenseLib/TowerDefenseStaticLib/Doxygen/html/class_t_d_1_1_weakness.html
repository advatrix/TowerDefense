This class describes Weakness effect Weakness affects enemy's Damage Multiplier so that the enemy will get higher damage from towers while being affected by Weakness.

Reimplementation of the update() function from the parent Effect class. If called firstly, changes the enemy's damageMultiplier according to the instance's "value" parameter, sets the "active" flag to true, and decrements remaining time. Otherwise only decrements remaining time.
